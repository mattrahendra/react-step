Panduan Setup Todo App

Prasyarat

  • Laptop dengan Node.js LTS (v20.x.x) [Download]
  • VS Code (atau editor lain) + ekstensi JavaScript
  • Smartphone Android/iOS dengan Expo Go (Play Store/App Store)
  • Koneksi Wi-Fi (laptop & HP di jaringan yang sama)
  • Waktu: 120 menit (60 menit setup & coding, 60 menit pengujian)
Catatan Teknis:
- Expo SDK 51, template blank JavaScript
- Tugas terlambat: 15 Mei 2025, 07:56 WIB
- Kode diuji: 10 Mei 2025, kompatibel Expo Go
  1. Persiapan Lingkungan
    • Buka terminal di VS Code (Ctrl + ~ atau Terminal > New Terminal).
    • Cek Node.js & npm: 
      node -v
npm -v
      Jika command not found, install Node.js LTS dari nodejs.org lalu restart terminal.
    • Install Expo CLI: 
      npm install -g expo-cli
      Jika gagal (izin ditolak), coba sudo npm install -g expo-cli (Mac/Linux) atau terminal sebagai admin (Windows). 
      expo --version
    • Pastikan Expo Go di HP sudah terinstall.
    • Buat folder proyek: 
      mkdir TodoApp
cd TodoApp
    Checkpoint: Node.js, Expo CLI, dan Expo Go siap? Lanjut ke langkah berikutnya!
  2. Membuat Proyek Expo
    • Buat proyek baru: 
      npx create-expo-app TodoApp
      Pilih blank (JavaScript) saat diminta.
    • Masuk ke folder proyek: 
      cd TodoApp
    • Jalankan aplikasi: 
      npx expo start
      Jika QR code tidak muncul, coba npx expo start --clear atau pastikan port 19000 tidak diblokir.
    • Buka Expo Go di HP, scan QR code.
    Struktur Awal:
    TodoApp/
    ├── App.js
    ├── app.json
    ├── package.json
    ├── node_modules/
    └── assets/
    Checkpoint: Layar default Expo muncul di HP? Lanjut ke install dependensi!
  3. Install Dependensi
    • Pastikan di folder TodoApp: 
      pwd
    • Install dependensi: 
      npm install @react-navigation/native @react-navigation/stack
npx expo install react-native-gesture-handler react-native-reanimated react-native-screens react-native-safe-area-context @react-native-community/masked-view
npx expo install @react-native-async-storage/async-storage
npx expo install @react-native-community/datetimepicker
    • Buat folder kode: 
      mkdir screens components styles
    Checkpoint: Dependensi & folder siap? Lanjut ke coding aplikasi!
